This is just a small Angular project to demonstrate that I'm comfortable with the framework. I've also made it better looking using animejs, which is one of my favourite JavaScript libraries. I took the .GPX data from a long run and turned the route into an SVG, where I animate it using animejs. Right now the 'run stats' are just an angular data structure I manually populated, because Strava recently updated their subscription services which changed how their API worked.
